`context.md` を読み込み、業界トレンド分析を実行してください。
web search を積極的に活用して最新情報を収集すること。

## 分析内容

### 1. マクロトレンド（5項目）

以下の観点で業界に影響を与えるグローバルな潮流を分析:
- 経済動向（景気、金利、為替など）
- 規制・政策（法改正、業界規制、補助金など）
- テクノロジー（AI、クラウド、モバイルなど該当するもの）
- 社会変化（人口動態、働き方、価値観の変化）
- 環境・サステナビリティ

### 2. ミクロトレンド（5〜7項目）

直近12ヶ月で業界内に見られる具体的なパターン:
- 新しいビジネスモデルの台頭
- 顧客行動の変化
- 技術の実用化・普及
- プレイヤーの動向（新規参入、撤退、統合）
- 価格や流通チャネルの変化

### 3. テクノロジーディスラプション

- 業界を変える可能性のある技術とその成熟度
- メインストリーム到達の予測時期
- 自社への影響（脅威 or 機会）

### 4. 投資シグナル

web search で以下を調査:
- 直近のVC投資・資金調達の動向
- M&A案件
- IPOや上場企業の業績動向
- 注目されている領域（投資家の関心がどこに向いているか）

### 5. タイムライン整理

各トレンドを以下の時間軸で分類:
- 短期（0〜1年）: 今すぐ対応が必要
- 中期（1〜3年）: 戦略に組み込むべき
- 長期（3〜5年）: ウォッチすべき

### 6. 「So What」分析

各トレンドについて:
- インパクト度（1〜10）
- 自社にとっての意味（具体的なアクションにつながる示唆）

## 出力

結果を `output/04-trends.md` に保存してください。

### フォーマット

- 冒頭にエグゼクティブサマリー（業界の方向性を3文で）
- 各トレンドはインパクト度付きで記載
- タイムラインは一覧テーブルで整理
- 最後に「自社が今注目すべきトレンド Top 3」を明示
