`context.md` を読み込み、市場規模（TAM）分析を実行してください。

## 分析内容

### 1. トップダウン分析

- グローバル市場規模から出発し、地域→業種→セグメントと絞り込む
- 各段階の絞り込み根拠を明示する
- 利用可能な市場調査レポート（Statista, Grand View Research, IDC, 矢野経済研究所など）を web search で探し、引用する

### 2. ボトムアップ分析

- ユニットエコノミクス × 想定顧客数で算出
- 想定顧客数の推計ロジックを明記する
- context.md のビジネスモデル情報をもとに単価を設定

### 3. TAM / SAM / SOM

- **TAM**（Total Addressable Market）: 理論上の最大市場
- **SAM**（Serviceable Addressable Market）: 実際にアプローチ可能な市場
- **SOM**（Serviceable Obtainable Market）: 現実的に獲得できる市場
- 各数値を金額と顧客数の両方で提示

### 4. 成長予測

- 今後5年間のCAGR（年平均成長率）を推計
- 成長ドライバーと阻害要因をそれぞれ列挙
- 楽観/中立/悲観の3シナリオで予測

### 5. 前提条件と限界

- すべての仮定を明示的にリストアップ
- データの信頼度を「高/中/低」で評価
- 追加調査が必要な領域を指摘

## 出力

結果を `output/01-tam.md` に保存してください。

### フォーマット

- 冒頭にエグゼクティブサマリー（5行以内）
- 各セクションは見出し付きで構造化
- 数値はテーブルで整理
- 最後に「次のステップ」として、この分析を踏まえて追加で行うべき調査を提案
